## Примери - теория
**1.** Еквивалентни ли са следните две програми?
```c++
int main()
{
    int grade;
    std::cin >> grade;
    if (grade == 2)
        std::cout << "fail";
    else if (grade == 5)
        std::cout << "nice";
    else if (grade == 6)
        std::cout << "excellent";
    else
        std::cout << "error";
}
```

```c++
int main()
{
    int grade;
    std::cin >> grade;
    switch(grade)
    {
        case 2: std::cout << "fail"; break;
        case 3:
        case 4:
        case 5: std::cout << "nice"; break;
        case 6: std::cout << "excellent"; break;
        default: std::cout << "error";
    }
}
```
**2.** Какво ще изведе на конзолата следния код?
```c++
int main()
{
    int number = 2;
    
    if (number < 2)
        std::cout << "first";
    else if (number = 0)
        std::cout << "second";
    else if (number >= 2)
        std::cout << "third";
    else
        std::cout << "unknown";
}
```
